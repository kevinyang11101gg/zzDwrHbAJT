# Python校园二手书交易系统简介

## 核心功能
- 登录注册
- 商品分类
- 商品详情
- 加入购物车
- 下单
- 评论
- 个人中心
- 商品管理
- 用户管理
- 订单管理
- 商品分析图

## 适用场景
- 校园环境中学生之间的二手书交易
- 促进校园资源共享，减少浪费

## 技术特点
- 使用Django框架进行开发，具有良好的扩展性和稳定性
- 基于MySQL数据库进行数据存储，保证数据安全
- 前端采用HTML、CSS和JavaScript，界面友好，交互流畅
